# Match3Mobile - Android-игра в жанре «3 в ряд»

[📦 Скачать APK](https://disk.yandex.ru/d/s5UghwCdcutn0g)
> **Внимание:** это демо‑версия для ознакомления. Перед установкой убедитесь, что в настройках устройства разрешены установки из неизвестных источников.

⚙️ Описание:
- Система начисления очков;
- Возможность строить разные комбинации;
- Бомба — специальный элемент, взрывающий окружающие ячейки (создаётся при определённой комбинации);
- Два режима заданий уровня (сбор указанной суммы очков и сбор ячеек указанного цвета);
- Звуковое сопровождение и фоновая музыка;
- UI на основе Canvas;
- System Input для обработки касаний;
- VContainer для внедрения зависимостей (DI);
- Пул объектов для переиспользования игровых элементов;
- Анимация через DOTween — плавные и оптимизированные эффекты перемещения.
  
⚙️ Оптимизация памяти:
- Проведено профилирование на реальном Android-устройстве с помощью Memory Profiler для анализа использования памяти;
- Было обнаружено высокое потребление из-за аудио и настроек рендера;
- В результате было сжало аудио и оптимизированы текстуры;
- Память снижена до ~350 MB на устройстве.

⚙️ Среды разработки:
- Visual studio 2026
- Unity 6000.3.10f
