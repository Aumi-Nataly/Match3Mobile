# Match3Mobile - Android-игра в жанре «3 в ряд»

- Система начисления очков;
- Возможность строить разные комбинации;
- Бомба — специальный элемент, взрывающий окружающие ячейки (создаётся при определённой комбинации);
- Два режима заданий уровня (сбор указанной суммы очков и сбор ячеек указанного цвета);
- Звуковое сопровождение и фоновая музыка;
- UI на основе Canvas;
- System Input для обработки касаний;
- VContainer для внедрения зависимостей (DI);
- Пул объектов для переиспользования игровых элементов;
- Анимация через DOTween — плавные и оптимизированные эффекты перемещения;
